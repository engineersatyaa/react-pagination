## API Reference

#### Get Items

```http
  GET https://jsonplaceholder.typicode.com/comments?_page=${currentPage}&_limit=${itemsLimit}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `currentPage` | `number` | **Required**. |
| `itemsLimit` | `number` | **Required**. |

- Responsive design.
